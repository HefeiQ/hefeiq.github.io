The University of Massachusetts, Boston

Computer Architecture and Organization with Labs
CS 341
Fall 2022, Spring & Fall 2023: Teaching Fellow

This course discusses computer architecture and organization from the embedded system point of view. It introduces computer hardware concepts and hardware-level programming for C programmers. Topics include Intel processor architecture, I-386 assembly language programming, serial and parallel port interfacing, programming I/O, interrupt programming, embedded system software development techniques and tools, digital logic circuits, etc.
